SVG Shape

The idea here is to only use the d attribute of the svg shape and cache the filters, templates and gradients so loading an svg is almost as cheap as loading a string

Avilable options and defaults

radius : 50
strokeWidth : 1
ringStrokeWidth : 0
ringStroke : "#777"
ringFill : "#777"
shape : "cube"
fill : "rgba(92, 69, 84, 0.2)"
stroke : "#777"
viewBox : "0 0 64 64"
